# Ballonspillet

Et interaktivt dansk stavningsspil hvor du skal stave ord for at åbne døre og komme gennem rum.

## Deploy til Vercel

### Metode 1: Via Vercel Web Interface (Nemmes)

1. Gå til [vercel.com](https://vercel.com) og log ind (brug GitHub/GitLab/Bitbucket)
2. Klik på "Add New Project"
3. Vælg "Upload" eller "Browse"
4. Vælg hele `Ballonspillet` mappen
5. Klik "Deploy"

### Metode 2: Via GitHub (Anbefalet)

1. Opret et GitHub repository
2. Upload alle filerne til GitHub
3. Gå til Vercel og klik "Add New Project"
4. Importer GitHub repository
5. Klik "Deploy" (ingen build settings nødvendige for static sites)

### Metode 3: Via Vercel CLI

Hvis du har Node.js installeret:

```bash
npm i -g vercel
vercel
```

## Spilfunktioner

- Vælg selv ord eller brug random 15 ord
- 3 liv system
- Tastatur support
- Visuelle effekter og animationer
- 44 danske ord inkluderet
